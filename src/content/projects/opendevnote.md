A comprehensive productivity suite built with Flutter for cross-platform deployment. Features an integrated scientific calculator with history, GitHub integration with repository overview, commit history, and GitHub Actions workflow runs. Issues and pull requests support coming in the next major update. Also includes a calendar view and a full-featured markdown editor with live preview and export capabilities.
