A lightweight terminal-based IRC client built with Rust and Ratatui. Features full IRC protocol support including SSL/TLS connections and multi-server support. The clean TUI interface provides an efficient IRC experience while maintaining a tiny binary size and minimal memory footprint suitable for servers and low-resource environments.
